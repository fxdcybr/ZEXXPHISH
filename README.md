# ZEXXPHISH

### Advanced Phishing Analyzer 👾

A lightweight Python-based phishing analysis tool for detecting suspicious URLs and generating professional PDF reports.

## Features

- URL Phishing Analysis
- Risk Score Calculation
- Threat Level Detection
- Scan History
- Professional PDF Report Generation
- Cyberpunk Terminal Interface
- Windows & Kali Linux Support

---

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ZEXXPHISH.git
cd ZEXXPHISH
```

---

## Running on Windows

### Step 1 - Create a Virtual Environment

```bash
python -m venv venv
```

### Step 2 - Activate the Virtual Environment

```bash
venv\Scripts\activate
```

### Step 3 - Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4 - Run ZEXXPHISH

```bash
python zexxphish.py
```

---

## Running on Kali Linux

### Step 1 - Create a Virtual Environment

```bash
python3 -m venv venv
```

### Step 2 - Activate the Virtual Environment

```bash
source venv/bin/activate
```

### Step 3 - Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4 - Run ZEXXPHISH

```bash
python3 zexxphish.py
```
